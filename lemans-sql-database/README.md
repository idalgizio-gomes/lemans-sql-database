## Sobre o projeto

Trabalho de Armazenamento e Acesso a Dados. Modelo relacional das 24 Horas de Le Mans com 7 entidades, **normalização até à 3ª forma normal**, resolução de relação muitos-para-muitos e scripts SQL de criação e povoamento.

## O que demonstra

- Modelação de dados (diagrama entidade-relação)
- Normalização de tabelas (3FN) e integridade referencial
- SQL (DDL/DML) e queries analíticas

## Organização do repositório

```
sql/      (scripts de criação e dados)
docs/     (relatório, diagrama ER)
```
